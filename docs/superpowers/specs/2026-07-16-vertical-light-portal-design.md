# 수직 빛기둥 이동 포털 설계

## 목표

기존 직사각형 안내 버튼을 고전 한국 MMORPG 분위기의 원형 워프 포털로 교체한다. 바닥 원형 마법진이 유일한 광원이며, 청백색 빛과 입자가 마법진에서 수직 위로 솟아 상단에서 자연스럽게 사라져야 한다.

## 시각 구성

- 바닥: 회전하는 이중 원형 마법진과 밝은 중심광
- 중간: 바닥에서 시작해 위로 갈수록 옅어지는 겹빛 기둥
- 입자: 작은 픽셀 광점이 아래에서 위로 이동하며 사라짐
- 라벨: 목적지 이름을 포털 아래에 항상 표시
- 선택 상태: 이동 대상 포털은 청백색에서 금빛으로 변하고 광량이 증가

위에서 내려오는 빛, 공중의 광원, 문 모양 프레임, 가로 소용돌이는 사용하지 않는다. 생성된 포털 원화의 바닥 발광원·수직 상승 실루엣을 구현 기준으로 사용하되, 런타임에서는 해상도와 맵 크기에 안정적인 CSS 레이어로 재현한다.

## 구조와 동작

- 기존 `button.world-portal`의 위치, 크기, 클릭 이벤트, 경로 탐색, 도착 판정과 페이드 전환은 유지한다.
- 버튼 안에 장식 전용 `world-portal__effect`와 접근성 이름을 담당하는 `world-portal__label`을 추가한다.
- 장식 레이어는 `aria-hidden="true"`이고 포인터 이벤트를 받지 않는다.
- 포털 빛은 버튼 위쪽으로 넘칠 수 있지만 실제 클릭 영역은 기존 월드 좌표 사각형과 동일하다.
- `prefers-reduced-motion`에서는 회전·상승 애니메이션을 멈추고 정적 발광 상태를 유지한다.

## 검증

- 컴포넌트 테스트로 장식 구조, 라벨 접근성, 기존 `z-index: 9000`과 이동 동작을 확인한다.
- 스타일 테스트로 바닥 링, 아래에서 위로 흐르는 빛기둥·입자, 대상 상태 및 모션 축소 규칙을 확인한다.
- 데스크톱과 모바일 화면에서 포털 위치, 라벨 겹침, 캐릭터 접근 및 전환 동작을 브라우저로 확인한다.
