# 조이스틱 포털 진입 설계

## 목표

조이스틱으로 포털 접근 타일에 진입하면 포털 클릭 자동 보행과 동일한 목적 맵으로 즉시 전환한다.

## 동작

- 조이스틱 터치 또는 조이스틱 키보드 방향 입력으로 포털의 `approach` 좌표에 진입하면 전환한다.
- 전환 목적지는 해당 포털의 `to`, 시작 좌표는 `spawn`을 사용한다.
- 전환 시 기존 `moveToZone`을 사용해 위치, 카메라, 실시간 상태, 이동 상태를 한 번에 초기화한다.
- 조이스틱이 계속 눌려 있어도 새 맵에서 연속 이동하지 않도록 외부 조이스틱 벡터를 `0,0`으로 초기화한다.
- 일반 맵 클릭 이동은 기존 동작을 유지한다.

## 검증

- 우리 집에서 조이스틱 키보드 입력으로 `home-to-neighborhood` 접근 타일에 도착했을 때 `neighborhood`로 전환되는지 테스트한다.
- 포털 접근 전에는 현재 맵을 유지하는지 확인한다.
- 목적 맵의 `spawn`으로 이동하고 도착 상태를 표시하는지 확인한다.
- 기존 포털 클릭, 맵 클릭, 미니맵, 조작 UI 클릭 테스트가 계속 통과해야 한다.
