# 웨이브 헤어 투명 배경 보정 설계

## 목표

웨이브 헤어 여성 하객의 컬 안쪽과 머리 외곽에 남은 흰 배경 픽셀을 투명하게 만들되, 얼굴 피부색과 흰색·밝은색 의상 및 액세서리는 보존한다.

## 범위

- `feminine-long-wave-dress`, `feminine-lavender-jacket-dress`, `feminine-teal-modern-hanbok`의 배포용 보행·대기 시트를 보정한다.
- 현재 3등신 소스 PNG는 수정하지 않고 `generate-character-assets.mjs`가 메모리에서 보정한 결과만 배포 경로에 기록한다.
- 머리 틈 판정은 `scripts/lib/guestHairBackground.mjs`에 두어 저작 단계와 배포 생성 단계가 같은 규칙을 사용한다.
- 원안 비율, 프레임 크기, 보행 순서와 색상은 변경하지 않는다.

## 처리 규칙

1. 저작 단계에서는 외곽과 연결된 밝은 무채색 배경을 기존 경계 플러드 필로 제거한다.
2. 저작·배포 단계 모두 외곽 머리 영역의 밝은 무채색 연결 성분을 검사한다.
3. 크기가 제한적이고 어두운 머리 픽셀에 둘러싸였으며 피부 경계가 없는 성분만 머리 틈으로 판정한다.
4. 기존보다 큰 웨이브 틈과 회색 안티앨리어싱 매트를 포함하도록 허용 크기와 머리 높이 범위를 확대한다.
5. 중앙 얼굴·눈·의상 영역과 피부에 접한 성분은 제거하지 않는다.

## 검증

- 큰 웨이브 틈과 회색 가장자리가 제거되는 실패 회귀 테스트를 추가한다.
- 기존 피부 보존 및 흰 의상 보존 테스트를 함께 통과해야 한다.
- 세 배포용 프리셋 시트를 마젠타 배경에 합성해 흰 잔여 픽셀을 육안 확인한다.
- 배포 생성 전후 소스 프레임이 바이트 단위로 동일한지 확인한다.
- 생성된 모든 캐릭터 에셋 감사와 전체 테스트를 통과해야 한다.
