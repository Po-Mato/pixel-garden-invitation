# 수직 빛기둥 이동 포털 구현 계획

> **실행 방식:** 현재 세션에서 테스트 우선으로 순차 실행한다. 사용자 요청에 따라 커밋과 푸시는 하지 않는다.

**목표:** 모든 월드 이동 포털을 바닥 원형 마법진에서 수직으로 빛이 솟는 애니메이션 포털로 교체한다.

**구조:** 기존 포털 버튼과 이동 상태는 유지하고 장식 전용 DOM 레이어만 추가한다. CSS의 바닥 링, 겹빛 기둥, 상승 입자와 대상 상태가 독립적으로 애니메이션된다.

**기술:** React, TypeScript, CSS keyframes, Vitest, Testing Library
