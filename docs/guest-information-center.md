# 하객 공지·FAQ 통합 운영 센터

## 접속

- 관리자 화면: `?admin=guest-info`
- 기존 관리자 비밀번호로 로그인합니다.
- 참석 현황 관리자 화면의 `공지·FAQ` 메뉴에서도 이동할 수 있습니다.

## 하객 화면

공지·FAQ는 다음 세 화면에서 같은 내용으로 제공됩니다.

- 입장 화면의 `공지·FAQ` 버튼
- 간편 초대장 상단의 알림 버튼
- 게임 화면 하단의 `공지·FAQ` 버튼

공개 상태인 FAQ와 현재 공개 기간에 포함된 공지만 표시됩니다. 공개할 내용이 하나도 없으면 버튼도 숨겨집니다.

## 공지 운영

공지는 제목, 본문, 중요도, 공개 여부, 상단 고정, 공개 시작·종료 시각, 행동 버튼, 정렬 순서를 설정할 수 있습니다.

- 중요도: 일반, 중요, 긴급
- 행동 버튼: 길 찾기, 예식장 전화, 외부 HTTPS 링크
- 공개 시작 전 공지는 `공개 예약`, 종료 시각이 지난 공지는 `공개 종료`로 표시됩니다.
- 공개 기간을 비워 두면 별도의 기간 제한 없이 노출됩니다.
- 상단 고정 공지는 일반 공지보다 먼저 표시됩니다.
- 하객 기기에서는 같은 공지를 한 세션에 한 번만 조회로 기록합니다.

## FAQ 운영

FAQ는 카테고리, 질문, 답변, 공개 여부, 추천 질문, 정렬 순서를 설정할 수 있습니다. 카테고리별로 묶여 하객 화면에 표시됩니다.

초기 데이터에는 다음 항목이 등록됩니다.

- 주차: 2시간 무료, 약 500대 이상
- 대중교통: 소사역 1번 출구에서 도보 약 3분
- 예식장: MJ컨벤션 5층 파티오볼룸
- 도착 권장: 예식 20~30분 전

## 운영 확인

관리자 상단 현황에서 전체 공지, 현재 공개 공지, 긴급 공지, 공개 FAQ, 누적 공지 조회 수를 확인할 수 있습니다. 수정이나 공개 상태 변경 후 하객 화면에는 최대 1분의 캐시 시간이 지나 반영될 수 있습니다.

공개 전에는 다음을 확인합니다.

1. 예약 공지의 시작·종료 시각이 한국 시간 기준으로 맞는지 확인합니다.
2. 외부 링크는 `https://` 주소로 입력합니다.
3. 긴급 공지는 꼭 필요한 기간에만 공개합니다.
4. 삭제는 되돌릴 수 없으므로 비공개 전환을 우선 사용합니다.
