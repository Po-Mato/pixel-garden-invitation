# 모바일 청첩장 실데이터 일괄 입력 체크리스트

이 문서는 임시 콘텐츠를 실제 정보로 한 번에 교체하기 위한 수집 목록입니다. 계좌번호와 전화번호는 공개 청첩장에 표시될 정보이므로 공개에 동의한 값만 전달합니다. 관리자 비밀번호와 Worker 비밀키는 이 문서에 기록하지 않습니다.

관리자 로그인 후 `?admin=content`에서 연락처·계좌·소개·스토리·공유 문구를 입력하고, `?admin=gallery`에서 실제 사진을 교체합니다. 각 화면에서 미리보기를 확인한 뒤 공개하며, 저장한 초안은 방문자에게 노출되지 않습니다. 공개 이력에서 이전 버전을 선택해 초안으로 복구할 수 있습니다. 공개 후에는 `?admin=readiness`에서 실데이터 입력 현황과 RSVP·방명록 운영 상태를 함께 점검합니다.

사진 관리 화면은 브라우저에서 원본을 자르고 640px·1024px WebP 파일로 변환한 뒤 비공개 저장소에 업로드합니다. 원본 파일과 촬영 정보는 서버에 저장하지 않습니다. 열 장의 사진과 대체 텍스트가 모두 준비된 초안만 공개할 수 있으며, 공개하기 전에는 기존 임시 사진이 계속 표시됩니다.

은행명·계좌번호·예금주가 모두 입력된 계좌만 표시되며, 간편송금 링크는 HTTPS 주소만 허용됩니다.

## 4. 실제 웨딩 사진 10장

현재 이미지는 임시 생성본입니다. 아래 슬롯과 방향에 맞춰 실제 사진을 준비한 뒤 `?admin=gallery`에서 순서대로 교체합니다.

- [ ] `01-cover-source`: 가로, 대표 사진 및 링크 미리보기
- [ ] `02-dress-bouquet-source`: 세로, 신부 소개 사진
- [ ] `03-side-walk-source`: 세로, 첫 인사 이야기
- [ ] `04-bench-silhouette-source`: 세로, 함께한 시간 이야기
- [ ] `05-veil-flowers-source`: 세로, 갤러리 사진
- [ ] `06-hands-rings-source`: 가로, 결혼 약속 이야기
- [ ] `07-bouquet-still-source`: 세로, 갤러리 사진
- [ ] `08-under-tree-source`: 가로, 갤러리 사진
- [ ] `09-garden-aisle-source`: 가로, 결혼식 이야기
- [ ] `10-sunlit-finale-source`: 세로, 갤러리 마무리 사진

사진마다 함께 정할 항목:

- 대체 텍스트: 사진에 실제로 보이는 내용을 설명
- 짧은 캡션: 필요한 사진만 입력
- 공개 동의 및 얼굴 노출 범위 확인

사진은 JPG·PNG·WebP 형식, 장당 최대 15MB까지 선택할 수 있습니다. 자르기 화면에서 확대와 좌우·상하 초점을 조절하고, 대체 텍스트와 필요한 캡션을 입력한 뒤 초안을 저장합니다. 열 장을 모두 교체한 다음 전체 미리보기를 검토하고 공개 반영합니다.

## 5. 신랑·신부 소개 문구

- [ ] 신랑 개인 소개 또는 인사말
- [ ] 신부 개인 소개 또는 인사말
- [ ] 두 사람이 함께 전하는 한 문장

확정된 문구는 `?admin=content`의 `소개·스토리` 탭에서 입력합니다.

## 6. 네 단계 스토리

- [ ] 첫 만남: 제목, 본문, 시기 또는 날짜
- [ ] 함께한 시간: 제목, 본문, 시기 또는 날짜
- [ ] 결혼을 약속한 순간: 제목, 본문, 시기 또는 날짜
- [ ] 결혼식으로 전하는 말: 제목, 본문

각 단계에서 사용할 사진 번호도 함께 확정합니다.

제목과 본문은 `?admin=content`에서 편집합니다. 타임라인의 사진 슬롯은 고정되어 있으므로 `?admin=gallery`에서 해당 번호의 사진을 교체하면 공개 화면에 함께 반영됩니다.
