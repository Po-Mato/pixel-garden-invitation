# 모바일 청첩장 실데이터 일괄 입력 체크리스트

이 문서는 임시 콘텐츠를 실제 정보로 한 번에 교체하기 위한 수집 목록입니다. 계좌번호와 전화번호는 공개 청첩장에 표시될 정보이므로 공개에 동의한 값만 전달합니다. 관리자 비밀번호와 Worker 비밀키는 이 문서에 기록하지 않습니다.

관리자 로그인 후 `?admin=readiness`에서 실데이터 입력 현황과 RSVP·방명록 운영 상태를 함께 점검할 수 있습니다. 실제 사진과 문구를 교체할 때는 `shared/src/weddingContent.ts`의 `weddingContentPublication` 상태도 `ready`로 변경해야 공개 준비 완료에 반영됩니다.

- [ ] 신랑 아버지: 성함, 전화번호
- [ ] 신랑 어머니: 성함, 전화번호

## 2. 신부 측 연락처

- [ ] 신부 아버지: 성함, 전화번호
- [ ] 신부 어머니: 성함, 전화번호

휴대전화 번호는 전화와 문자 버튼이 함께 표시됩니다. 유선전화는 전화 버튼만 표시됩니다.

## 3. 마음 전하실 곳

아래 여섯 대상별로 정보를 준비합니다.

- [ ] 신랑 이승재
- [ ] 신랑 아버지
- [ ] 신랑 어머니
- [ ] 신부 이건희
- [ ] 신부 아버지
- [ ] 신부 어머니

각 대상의 입력 항목:

- 은행명
- 계좌번호
- 예금주
- 카카오페이 송금 URL, 선택
- 토스 송금 URL, 선택

은행명·계좌번호·예금주가 모두 입력된 계좌만 표시되며, 간편송금 링크는 HTTPS 주소만 허용됩니다.

## 4. 실제 웨딩 사진 10장

현재 이미지는 임시 생성본입니다. 아래 파일명과 방향에 맞춰 실제 사진을 준비합니다.

- [ ] `01-cover-source`: 가로, 대표 사진 및 링크 미리보기
- [ ] `02-dress-bouquet-source`: 세로, 신부 소개 사진
- [ ] `03-side-walk-source`: 세로, 첫 인사 이야기
- [ ] `04-bench-silhouette-source`: 세로, 함께한 시간 이야기
- [ ] `05-veil-flowers-source`: 세로, 갤러리 사진
- [ ] `06-hands-rings-source`: 가로, 결혼 약속 이야기
- [ ] `07-bouquet-still-source`: 세로, 갤러리 사진
- [ ] `08-under-tree-source`: 가로, 갤러리 사진
- [ ] `09-garden-aisle-source`: 가로, 결혼식 이야기
- [ ] `10-sunlit-finale-source`: 세로, 갤러리 마무리 사진

사진마다 함께 정할 항목:

- 대체 텍스트: 사진에 실제로 보이는 내용을 설명
- 짧은 캡션: 필요한 사진만 입력
- 공개 동의 및 얼굴 노출 범위 확인

원본 교체 절차는 `wedding-photo-sources/README.md`를 따릅니다.

## 6. 네 단계 스토리

- [ ] 첫 만남: 제목, 본문, 시기 또는 날짜
- [ ] 함께한 시간: 제목, 본문, 시기 또는 날짜
- [ ] 결혼을 약속한 순간: 제목, 본문, 시기 또는 날짜
- [ ] 결혼식으로 전하는 말: 제목, 본문

각 단계에서 사용할 사진 번호도 함께 확정합니다.
